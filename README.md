Ender-3-Plotter is an open-source project aiming to convert a Creality Ender 3 into a 2D plotter for drawing, sketching, and printing blueprints.

Instead of buying an expensive plotter, the goal is to reuse as many original parts of the printer as possible to create a low-cost and accessible solution.

This project is fully open-source to give back to the community that helped me learn and experiment.
